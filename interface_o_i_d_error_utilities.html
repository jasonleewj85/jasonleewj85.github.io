◆ isOAuthErrorDomain:() [1/2]

+ (BOOL) isOAuthErrorDomain: (NSString *)  errorDomain

Returns true if the given error domain is an OAuth error domain.

Parameters
errorDomainThe error domain to test. An OAuth error domain is used for errors returned per RFC6749 sections 4.1.2.1 and 5.2. Other errors, such as network errors can also occur but they will not have an OAuth error domain.
See also
https://tools.ietf.org/html/rfc6749#section-4.1.2.1
https://tools.ietf.org/html/rfc6749#section-5.2
