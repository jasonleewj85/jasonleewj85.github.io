Detailed Description

A utility class for creating and parsing URL query components encoded with the application/x-www-form-urlencoded format. Supports application/x-www-form-urlencoded encoding and decoding, specifically '+' is replaced with space before percent decoding. For encoding, simply percent encodes space, as this is valid application/x-www-form-urlencoded.

See also
https://tools.ietf.org/html/rfc6749#section-4.1.2
https://tools.ietf.org/html/rfc6749#section-4.1.3
https://tools.ietf.org/html/rfc6749#appendix-B
https://url.spec.whatwg.org/#urlencoded-parsing
